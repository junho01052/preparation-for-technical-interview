GET, POST 방식의 차이

1. 용도
GET - 주로 서버에서 정보를 조회할 때 사용됩니다
게시판을 예로 들면 이미 서버에 저장된 사진이나 게시글을 불러올 때 사용됩니다
POST - 서버에 정보를 제출하거나 데이터를 전송할 때 사용됩니다
마찬가지로 게시판을 예로 들면, 새로운 게시글이나 새로운 회원정보를 서버에 등록할 때 사용합니다

2. 멱등성
GET은 멱등입니다 즉, 동일한 GET 요청을 여러 번 전송하더라도 결과는 항상 동일해야 합니다
POST는 멱등이 아닙니다 동일한 POST 요청을 여러 번 전송하면 서버의 상태가 여러 번 변경될 수 있습니다

이러한 차이점들은 각 HTTP 메서드의 기본 설계 의도와 관련이 있습니다
GET은 안전하고 멱등한 메서드로서 정보를 검색하는 데 사용되며,
POST는 서버의 상태를 변경하거나 데이터를 저장하는 데 사용됩니다
