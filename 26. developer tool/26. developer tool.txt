개발자 도구를 활용한 경험

브라우저의 개발자 도구를 사용하여 HTML, CSS, JavaScript 코드를 실시간으로 디버깅하고 수정하는 방법을 사용합니다
이를 통해 레이아웃 문제, 스타일 이슈 등을 해결할 수 있습니다

Console.log와 디버깅: JavaScript의 console.log 함수를 사용하여 코드의 특정 부분에서 어떤 값이 출력되는지 확인할 수 있습니다

네트워크 탭 활용: 개발자 도구의 네트워크 탭을 사용하여 API 호출이 제대로 이루어지고 있는지,
어떤 데이터가 주고받아지는지 등을 확인할 수 있습니다

Performance 탭을 활용한 성능 최적화: Performance 탭을 활용하여 웹사이트의 성능 문제를 진단하고, 
렌더링 이슈, 불필요한 리렌더링, 지연 로딩 등을 해결하기 위한 방법을 적용했습니다

접근성 체크: Accessibility 탭을 활용하여 웹사이트의 접근성 문제를 확인하고 적용했습니다

모바일 뷰 테스트: 개발자 도구에서 제공하는 모바일 뷰를 활용하여 반응형 디자인을 테스트했습니다

커스텀 스니펫 사용: 자주 사용하는 코드 스니펫을 개발자 도구에 저장해두고, 디버깅이나 테스트에 활용했습니다

실제 문제 해결 사례: CORS 에러 해결, application 탭을 활용한 JWT 요청을 확인했습니다
